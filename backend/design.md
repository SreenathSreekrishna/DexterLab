# Design for backend

## db design

1. **users** - uID, uName, uMail, uPwd 
2. **employees** - eID, eName, eMail, ePwd
3. **parts** - pID, pName, pDesc, pLabour, pStock, pNumber
4. **inventions** - iID, iName, iCreator, iParts
5. **analytics** - aID, type, name, numSold
6. **carts** - cID, uID, iID
